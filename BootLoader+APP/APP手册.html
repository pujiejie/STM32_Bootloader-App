3. UDS 服务全景

0x10 会话控制

子功能行为
0x01默认会话，重置安全锁
0x02编程会话 → 写暗号 + 软复位 → Bootloader
0x03扩展诊断会话，重置安全锁
10 01 → 50 01 00 32 27 10
10 02 → 50 02 00 32 → NVIC_SystemReset
10 03 → 50 03 00 32 27 10

0x11 ECU 复位

11 01 → 51 01 → hardReset  11 03 → 51 03 → softReset

0x14 清除诊断信息

14 FF FF FF → 54 // 清全部
14 00 F1 00 → 54 // 清 0xF100

0x19 读 DTC

受 g_dtc_enabled 控制，关闭后回 7F 19 22。

19 01 FF → 59 01 00 00 + count  // 数量
19 02 FF → 59 02 00 00 + DTC[3]+status // 列表
19 06    → 59 06 + 全部DTC      // 全部

0x22 读 BMS 数据

DID内容DID内容
0xF190VIN 车架号0xF100总电压
0xF189软件版本0xF101电流
0xF102SOC0xF103SOH
0xF104/05单体电压max/min0xF106/07温度max/min
0xF108充放电状态0xF109电芯数量
0xF110CAN 帧计数0xF111DTC 数量
0xF180充电开关0xF181均衡开关
22 F1 02 → 62 F1 02 4E  // SOC=78%
22 F1 10 → 62 F1 10 xx xx xx xx // CAN 计数

0x27 安全访问

写参数 (0x2E) 和 例行程序 (0x31) 需要先解锁

27 01 → 67 01 + seed[4大端]
// key = ((~seed)^0x5A5A5A5A)+0x3618BC91
27 02 + key[4大端] → 67 02  // 解锁
// 3次失败 → 锁10秒

0x28 通信控制

28 00 00 → 68 00 00 // 正常
28 01 03 → 68 01 03 // 静默 (停发NM)

0x2E 写参数（需解锁）

先发 27 01 + 27 02 解锁，再发 2E

DID请求效果
0xF1022E F1 02 50SOC = 50%
0xF1082E F1 08 01状态 = 充电
0xF1012E F1 01 05 DC电流 = 1500mA
0xF1802E F1 80 00禁用充电
0xF1812E F1 81 01开启均衡
// 未解锁就写 →
7F 2E 33  // NRC 33: 安全拒绝

0x31 例行程序（需解锁）

RID请求效果
0xE00131 01 E0 01BMS 自检
0xE00231 01 E0 02SOC = 100%
0xE00331 01 E0 03SOH = 100%

0x85 控制 DTC 设置

85 01 → C5 01 // 开启 DTC
85 02 → C5 02 // 关闭 DTC

6. NRC 否定响应码

NRC含义可能原因
0x10通用拒绝未分类错误
0x11服务不支持APP 不支持刷写服务
0x12子功能不支持未知子功能
0x13长度错误报文太短
0x22条件不满足DTC关闭/非编程会话/刷写去Bootloader
0x33安全拒绝2E/31 未解锁
0x35密钥无效27 02 密钥算错
0x36重试超限3次失败
0x37锁定延迟中等10秒
